**★Eine notwendige Bedingung für das Streaming über das externe Netzwerk: eine öffentliche IP-Adresse.**

**Wer eine öffentliche IP-Adresse hat, kann weiterlesen.**

### Szenario 1

Markenrouter mit offiziellem System
Kehren Sie zur Software-Homepage zurück – wählen Sie die rot umkreiste Option.

![](public/images/air3/ps5-wan-1.PNG)

* Öffnen Sie die rot umkreiste Option – Router konfigurieren.

![](public/images/air3/ps5-wan-2.png)

* Theoretisch sollte eine „erfolgreich“-Meldung erscheinen, oder es kann eine Fehlermeldung angezeigt werden.

![](public/images/air3/ps5-wan-3.png)

## Szenario 2

Für Router mit UPnP – aktivieren Sie einfach UPnP.

### Szenario 3

Manuelle Einrichtung von DDNS auf einem Router

Manuelle Konfiguration der Router-Portweiterleitung:

* UDP 9302

* UDP und TCP 9295

* UDP 9296

* UDP 9297

Alternativ leiten Sie alle Ports der PS5 weiter.

![](public/images/air3/ps5-wan-4.png)

**Das oben Genannte deckt alle Informationen zu den Einstellungen für das externe Netzwerk-Streaming ab.**

***Hinweis: Die Einstellungen für das externe Netzwerk-Streaming variieren je nach Router. Bitte wählen Sie die entsprechende Methode unter Verwendung der folgenden Schlüsselwörter: (Routername), (Portweiterleitung), (DDNS).***

### Streaming-Bereich

* Hauptschnittstelle - Startseite

![](public/images/air3/ps5-wan-5.png)

#### ①LOKALE VERBINDUNG

![](public/images/air3/ps5-wan-6.png)

* Wenn Sie lokal spielen möchten, beginnt die Suche nach der PS5. (Befindet sich die PS5 im Standby-Modus, wird sie zuerst aufgeweckt und dann verbunden.)

![](public/images/air3/ps5-wan-7.png)

#### ②REMOTE-VERBINDUNG

![](public/images/air3/ps5-wan-8.png)

* „Automatisch“ startet die automatische Suche nach der PS5.

![](public/images/air3/ps5-wan-9.png)

* Geben Sie Ihren DDNS-Domainnamen ein (muss nur einmal ausgefüllt werden).

![](public/images/air3/ps5-wan-10.png)

* Wählen Sie eine von zwei Optionen – wählen Sie basierend auf dem Zustand des Geräts.

![](public/images/air3/ps5-wan-11.png)
