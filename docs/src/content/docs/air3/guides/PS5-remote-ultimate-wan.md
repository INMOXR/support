**★There is a necessary condition for external network streaming: a public IP address.**

**Those with a public IP address can continue reading.**

### Scenario 1: Branded Router with Official SystemReturn to the software homepage - select the red-circled option

![](public/images/air3/ps5-wan-1.PNG)

* Open the option marked with a red circle - Configure Router

![](public/images/air3/ps5-wan-2.png)

* Theoretically, a "successful" prompt should appear, or an error prompt may show up.

![](public/images/air3/ps5-wan-3.png)


### **Scenario 2**: For the router with UPnP - simply enable UPnP.

### **Scenario 3**: Manually setting up DDNS on a router

Manual Configuration of Router Port Forwarding:

* UDP 9302

* UDP and TCP 9295

* UDP 9296

* UDP 9297

Alternatively, forward all ports of the PS5.

![](public/images/air3/ps5-wan-4.png)



**The above covers all the information about external network streaming settings.**

***Note: The external network streaming settings vary with each router. Please select the corresponding method using the following keywords: (router name), (port forwarding), (DDNS).***

* Main Interface - Home

#### ①LOCAL CONNECT

![](public/images/air3/ps5-wan-6.png)

![](public/images/air3/ps5-wan-8.png)

![](public/images/air3/ps5-wan-9.png)

* Enter your DDNS domain name (only needs to be filled in once)

![](public/images/air3/ps5-wan-10.png)

* Choose one out of two - select based on the condition of the machine
