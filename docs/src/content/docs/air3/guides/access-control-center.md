### **Background App Whitelist Mode**

You can add specific applications to a whitelist. Even if you perform a one-tap cache clear, whitelisted apps will not be cleared and will continue to run in the background.

![](public/images/air3/access-the-control-center-6.png)

Steps to add apps to the Whitelist Mode:

1. Navigate to Glasses Settings (Settings).

2. Select the **APPs** option.

3. Tap Background App Whitelist.

4. From this screen, you can select the applications you wish to add to the whitelist, preventing them from being cleared by one-tap or automatic background cleanup.
