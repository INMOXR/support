### パート 1 デバイス情報と設定  
  
① グラスのデスクトップで、[設定]アプリアイコンまでスワイプし、タップして設定インターフェースに入ります。  
  
  
![](public/images/air3/unlock-developer-1.png)  
  
  
② 「スマートグラスについて」をタップ → スマートグラスの基本情報（[デバイス名]、[システムバージョン]）を表示します。  
  
  
![](public/images/air3/unlock-developer-2.png)  
  
  
③ デバイス名 → タップしてデバイス名を変更します。  
  
  
### パート 2 開発者モードを有効にする  
  
① スマートグラスについて → デバイス識別子 [ビルド番号]。これは当社のスマートグラスのバージョン番号です。  
  
  
![](public/images/air3/unlock-developer-3.png)  
  
  
② [ビルド番号] を10回タップして開発者モードに入ります。  
  
  
![](public/images/air3/unlock-developer-4.png)  
  
  
注：専門家以外の方が開発者モードに入るためにこの操作を行うことは推奨されません。
