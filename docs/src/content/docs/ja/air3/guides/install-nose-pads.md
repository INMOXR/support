## 📦 同梱内容

Air 3の箱の中には、以下のものが含まれています：

* エアノーズパッド＆クッション

* 交換用ノーズパッド（3サイズ）：小（S）、中（M）、大（L）

これらのアクセサリーは、最適な快適性とフィット感を得るために、メガネの高さや角度を調整するのに役立ちます。



## 🔧 交換ステップバイステップガイド

### ステップ 1: 工具の準備


* 付属の小型ドライバー。

* 取り付けたい新しいノーズパッドとクッション。

* 小さな部品をなくさないように、清潔で明るい作業スペース。

### ステップ 2: 元のノーズパッドを取り外す


1. ゴム製のノーズクッションを優しく下に引き、中の小さなネジを露出させます。

2. 小型ドライバーを使って、ネジを反時計回りに回して緩めます。

   * コツ：ネジは非常に小さいので、落とさないように注意してください。

3. ネジを取り外したら、ノーズパッドのフレーム全体を取り出すことができます。

4. 古い部品は安全な場所に取っておきます。

### ステップ 3: 新しいノーズパッドを取り付ける


1. 新しいノーズパッドフレームをメガネの取り付け穴に合わせます。

2. ネジを挿入し、ドライバーで時計回りに回してしっかりと止まるまで締めます。

   * 締めすぎないでください。フレームが固定される程度に締めます。

3. エアノーズクッションを、柔らかい「カチッ」という音がするまでしっかりと押し込み、完全に装着されたことを確認します。

### ステップ 4: フィットと調整

* メガネをかけて、快適な位置に調整します。

* 必要に応じて、クッションを優しく引き離して取り外し、別のサイズを試すことができます。

## 💡 役立つヒント


* ネジや小さな部品は、トレイや布の上で扱うようにしてください。

* サイズを切り替える場合は、使用しないノーズパッドは元のパッケージに保管してください。

* クッションがしっかりと「カチッ」とはまることを確認し、使用中に外れないようにしてください。

さらにサポートが必要な場合は、サポートセンターをご覧になるか、カスタマーサービスチームまでお問い合わせください。
